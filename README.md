S. aureus AMR Surveillance Mobile Application
A dual-role, offline-first mobile application for real-time Antimicrobial Resistance (AMR) surveillance of Staphylococcus aureus, designed for low-resource clinical settings in Uganda.

Built to bridge the critical gap between microbiology laboratories and clinical decision-makers by providing frictionless access to Antimicrobial Susceptibility Testing (AST) reports and Machine Learning (ML) resistance predictions.

🎯 The Problem
In many low-resource settings, Antimicrobial Susceptibility Testing (AST) data remains trapped in local laboratory records. By the time aggregated resistance data reaches the clinicians prescribing antibiotics, it is often outdated. This leads to empirical therapy failures, as clinicians prescribe antibiotics based on historical guidelines rather than real-time local resistance patterns.

💡 The Solution
This platform enforces a strict Separation of Concerns architecture, recognizing that data producers (Lab Technicians) and data consumers (Clinicians) have completely different usability and security needs:

The Lab Channel (Authenticated Write-Access): A structured, offline-first 3-step workflow (Sample → Isolate → AST) that holds lab technicians accountable for the data they generate.
The Clinical Channel (Unauthenticated Read-Only): A zero-friction, no-login-required interface that gives clinicians instant access to AST reports and localized resistance statistics, removing barriers to evidence-based prescribing.
✨ Key Features
For Laboratory Personnel
3-Step Data Entry: Structured workflow to register specimens, identify S. aureus isolates, and record AST results (Zone diameter/mm + S/I/R interpretation).
Auto-MDR Flagging: System automatically flags Multi-Drug Resistant isolates if resistance is recorded across 3+ antibiotic classes.
Offline-First Architecture: Full data entry capability without internet connectivity. Data syncs seamlessly to the central server when connectivity is restored (using UUIDs to prevent collision).
Automated Reporting: Instant generation of standardized AST lab reports upon workflow completion.
For Clinicians / Physicians
Frictionless Access: No accounts, no passwords, no login delays. Tap "Continue as Clinician" to view data instantly.
Live AST Reports: View color-coded, standardized lab reports with local resistance context (e.g., "58.8% local resistance to this drug").
ML Resistance Forecasts: View 6-month resistance probability predictions generated by an external Python ML pipeline, complete with confidence intervals.
Historical Trends: Visualize multi-year resistance trends for S. aureus across all 25 tracked antibiotics.

Focus Organism: Staphylococcus aureus (Tracking 25 antibiotics: Penicillins, Cephalosporins, Carbapenems, Glycopeptides, Oxazolidinones, Macrolides, Aminoglycosides, Tetracyclines, Fluoroquinolones, Sulfonamides, Lincosamides, Phenicols).

🗄️ Database Schema Highlights
Normalized Workflow: samples (1) → isolates (Few) → susceptibility_tests (Many).
Pre-calculated Aggregates: Uses a resistance_aggregates table updated nightly/sync to ensure sub-second load times for clinician dashboards on slow 3G networks.
Offline-Safe Primary Keys: All tables use UUIDs instead of auto-incrementing integers to prevent sync collisions when multiple devices enter data offline.
Strict Privacy: No Protected Health Information (PHI) stored. Patient data limited to Age Group, Sex, and Ward.
